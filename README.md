## *Direct from Farm, Fair for All*


## Overview
AgriLink is an Ethiopia-based AgTech platform designed to connect smallholder farmers directly with urban consumers through a transparent, government-aligned agricultural supply chain. The project addresses inefficiencies in the agricultural value chain by reducing middlemen, stabilizing farmer incomes, improving logistics, and ensuring affordable, quality-assured food access for urban markets.

AgriLink combines simple digital technologies with physical infrastructure (branches, warehouses, logistics partnerships) to create a scalable and inclusive farmer-to-consumer ecosystem.

**Key Features**
* Direct farmer-to-consumer agricultural marketplace
* Transparent pricing aligned with government reference prices
* IVR-based farmer onboarding for non-smartphone users
* Digital consumer ordering and delivery system
* Integrated logistics, aggregation, and light processing workflows
* Scalable, branch-based operational model

**Target Users**
* Smallholder farmers in Ethiopian agricultural production zones
* Urban consumers (initially Addis Ababa)
* Logistics and transport partners
* Development partners, investors, and government stakeholders

**Roadmap**
* Integrate testimonial management system
* Add multilingual support (Amharic / English)
* Prepare backend APIs for farmer, order, and logistics data
* Integrate IVR and digital payment systems
* Expand UI for partner and investor dashboards

**Impact**
*AgriLink aims to:*
* Improve farmer income stability
* Reduce post-harvest losses
* Lower urban food prices
* Strengthen ruralâ€“urban supply chain integration
* Support sustainable and inclusive agricultural growth in Ethiopia

**## Getting Started (Local Development)**
**###Prerequisites**
* Node.js (v18 or later recommended)
* npm

**Installation & Run**
```sh
# Step 1: Clone the repository using the project's Git URL.
git clone https://github.com/Mulat-K/AgriLink.git

# Step 2: Navigate to the project directory.
cd AgriLink

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```
